# Difference between framework and library

We can say that a library implements a particular function. We can define a framework as a collection of libraries implementing a particular methodology. 

The developer writing the code is in charge of the application flow and the developer decides when to call the library. However, when we use a framework, the framework decides when to call the library. 

A library is a collection of reusable code. A framework is a collection of libs that implement a particular methodology.
